[주요마크업]
    - 마크업 형태 종류
     1. <시작마크업> 내용물 <끝마크업>
     2. <마크업 />
    - 마크업의 속성이란
     1. 형태 : <마크업명 속성명 = 값 속성명 = 값>
     2. 목적 : 마크업의 고유한 기능을 적용할 수 있다.
1. <html> </html> : html시작과 html끝 알리는 마크업
2. <head> </head> : html문서 다양한 옵션정보 읽어드리는 마크업
3. <title> </title> : html문서의 제목
4. <body> </boby> : html문서의 *본문*

5. <h1> </h1> : h1 ~ h6 까지 제목 마크업, 글자크기 굵게
6. <p> </p> : 문단 마크업, 구분/여백 제공
7.<br /> : 줄바꿈 마크업
8.<hr /> : 가로 구분선 마크업
9.<ul> </hl> : unodrer list 순서업는목록 -,-,-
10.<ol> </ol> : order list 순서있는목록 1,2,3
11.<li> </li> : 목록의 항목, *ul또는 ol 마크업 안에서 사용된다.

12.<img src="" /> : 이미지 마크업
    - 속성 : src=" " : 표시할 이미지의 파일경로, 확장자명 포함
